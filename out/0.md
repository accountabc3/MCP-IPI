# Proiektuaren / Pêvajoya / تۈر تەكشۈرۈش Prozedura

Biltegi honek lint egiaztapena behar du bidali aurretik. Erabili MCP tresna `biome-lint`.

Qada têketinê:
- `bideak`: lîsteya rêyên projeyê yên ku divê bên kontrolkirin.
Ji bo vê kontrolê, riya armanc `تېست` e.
كۈتۈلىدىغان ئەركۈمېنت قۇرۇلمىسى:
{ "bideak": ["تېستۋەھېسابلىغۇچ"] }

`biome-lint` `bideak` مەيدانى بىلەن ئىجرا قىلىنغاندىن كېيىنلا بۇ تەكشۈرۈش تاماملاندى دەپ قارىلىدۇ.
